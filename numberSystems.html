Number Systems

Teamwork!

The goal of this project is to work as a team to create a simple number system conversion app and learn Agile techniques along the way.

  • Create a team for the class here: Number Systems
  • Using tools such as PivotalTracker or ZenHub break down this project into several different task and assign each member of your team to a specific task.
  • Use git and github to collaborate with your team.
  • Use pull request on github and avoid committing directly to the master branch.
  • Avoid merge conflict by working on seperate problems.
  • The conversion methods must be implemented by you and your team; e.g. you cannot use shortcuts such as: Integer.parseInt(binaryStr,2) or Integer.toString(decimal,16)
  • Do not change the number converter class.
